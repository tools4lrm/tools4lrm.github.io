
# Anatomia e Semântica do Portal NossaReceita

Desenvolver para a web é como construir um edifício: antes da pintura e decoração (CSS), precisamos de uma fundação sólida e paredes bem alinhadas (HTML). Abaixo, detalhamos cada elemento utilizado em nosso projeto até o momento.

> **ATENÇÃO** : Todos os exemplos de código utilizados para representar os elementos poderão ser acessados e estudados, de como completo, [aqui](https://github.com/tools4lrm/tools4lrm.github.io/blob/0be10781cebf47852036cfedcdf7064b79289aca/index.html#L1-L125).

## 1. As Configurações 

Todo documento HTML profissional começa com instruções para o navegador. Então, cada elemento a seguir é uma instrução para o navegador:

`<!DOCTYPE html>`: É o sinalizador de que estamos usando o padrão `HTML5`. Ele garante que o navegador não tente interpretar seu código de maneira obsoleta ou restritiva (`XHTML`).

`<html lang="pt-br">`: Define o idioma. Isso é vital para a acessibilidade, pois informa aos softwares de leitura para cegos qual pronúncia utilizar.

`<head>` e `<meta charset="utf-8">`: O head descreve informações necessárias para o navegador. O `charset` configura a codificação, qual conjunto de caracteres estarão sendo utilizado no documento, para que palavras como "nutricional" ou "saudável" não fiquem com caracteres estranhos.

`<title>`: É o nome que aparece na aba do navegador. Essencial para que o usuário se localize e para que o Google encontre seu site.

``` html

<!-- o DOCTYPE avisa ao navegador que estamos usando o padrão HTML5. Sem ele, 
 o navegador pode NÃO sabe como interpretar, de modo específico, a estrutura 
 do nosso documento. Lembrem que a estruturação dos documentos HTMLs sofreram 
 mudanças ao longo do tempo -->
<!DOCTYPE html>

<!--  O elemento raiz que envolve tudo. O atributo lang="pt-br" 
 essencial para tecnologias assistivas saberem o idioma da página. -->
<html lang="pt-br">

    <!-- <head> é o local utilizado para passar informações para o navegador. Existe
     o conteúdo da página e as informações que passamos para o navegador, como o <meta charset="utf-8">, 
     que garante que acentos e caracteres especiais (como o "ç" e o til) sejam exibidos 
     corretamente. -->
    <head>
        <!-- O texto que aparece na aba do navegador. É vital para o SEO (mecanismo de busca) 
         e para o usuário encontrar sua página entre várias abas abertas.-->
        <title>NossaReceita</title>
        <meta charset="utf-8" />
    </head>

... 

```


## 2. A Hierarquia de Informação (Títulos e Parágrafos)

A semântica é o coração da web moderna. Não usamos tags apenas para mudar o tamanho da letra, mas para dar significado:

`<h1>` a `<h3>`: O `<h1>` é o título soberano da página. Os subtítulos (`<h2>`, `<h3>`) organizam os assuntos em ordem de importância. Isso ajuda os motores de busca a entenderem o "assunto principal" do seu site.

`<p>`: O parágrafo é o bloco básico de texto. Ele cria o espaçamento necessário para uma leitura confortável.

``` html

...

    <!-- Tudo o que deseja que o usuário visualize entra aqui. Aqui é o seu documento -->
    <body>

        <!-- O <h1> é o título principal (o nome do site), e o <h2> é o subtítulo explicativo. 
         A semântica aqui ajuda não apenas o design, mas também ferramentas de busca (como o 
         Google) a entenderem sobre o que é a página.-->
        <h1>NossaReceita</h1>
        <h2>Bem-vindo à NossaReceita, seu portal de cursos,
            dicas nutricionais e receitas para uma vida 
            saudável
        </h2>

        <!-- o <p> define os parágrafos de texto. Eles ajudam a criar respiros visuais e organizam 
         a leitura. -->
        <p>Entendemos que a jornada por um estilo de vida mais 
            equilibrado começa com informação de qualidade, 
            por isso, reunimos em um só lugar tudo o que você 
            precisa: desde guias práticos sobre nutrição até o 
            passo a passo detalhado de receitas que unem sabor e 
            saúde em cada detalhe.</p>

...

```

## 3. Elementos Vazios e Incorporação: A Tag `<img>`

Diferente dos parágrafos (`<p>`), a imagem é um elemento vazio (não possui conteúdo de texto interno nem tag de fechamento </img>).  

- Atributo `src`: É o caminho do arquivo. Note que utilizei `imagens/NossaReceita.png`. Isso indica que o navegador deve, **considerando a localização da página que contém o elemento** `<img>`. entrar na pasta "imagens" para buscar o arquivo de nome `NossaReceita` e de extensão `png` .  
- Atributo `alt` (O mais importante!): É o texto alternativo. Se a imagem não carregar ou se o usuário for deficiente visual, este texto comunicará o conteúdo da imagem. É um pilar da acessibilidade digital.  

``` html

...

        <!-- <img> é um elemento vazio (como explica o material Padrões WEB - Aula 05). 
         Ele não tem uma tag de fechamento (</img> não existe!) porque ele não "envolve" texto, 
         ele apenas incorpora um arquivo externo para dentro da página.

         - O atributo src (Source/Fonte) informa exatamente em qual pasta (imagens/), tendo como base a
         posição do arquivo em questão (index.hmtl), e o nome do arquivo (NossaReceita.png) que o navegador deve buscar.

         - O atributo alt é o texto alternativo.Elemento importante para a acessibilidade, pois para quem não vê, o 
         leitor de tela lerá em voz alta o valor (No meu caso, Logotipo do Site). Para quem tem conexão lenta, a imagem 
         falhar ao carregar, o texto "Logotipo do Site" aparecerá no lugar, mantendo a página contextualizada.

         -->
        <img src="imagens/NossaReceita.png" alt="Logotipo do Site" />

...

```

## 4. Organização de Dados: Listas e Tabelas

Para apresentar benefícios e agendas, utilizamos estruturas específicas:  

`<ul>` e `<li>`: Criam listas não ordenadas (com marcadores). São perfeitas para destacar diferenciais onde a ordem não é o foco. Para termos uma lista onde a ordem é necessária, basta trocar o elemento `<ul>` (_unordered list_) por `<ol>` (_ordered list_).  

``` html

...

    <!-- Usamos listas não ordenadas para itens onde a ordem não altera o sentido. É uma excelente 
     forma de listar benefícios, como os diferenciais da "NossaReceita". -->
    <ul>
        <li>Dicas práticas para o seu dia a dia.</li>
        <li>Receitas saudáveis e rápidas de preparar.</li>
        <li>Cursos exclusivos para iniciantes na cozinha.</li>
    </ul>

...

```

- `<table>`: É a estrutura para dados tabulares, organizando esses dados com linhas e colunas.
  - `<tr>`: Define cada linha da tabela.
  - `<th>`: Define o cabeçalho (títulos das colunas), que o navegador deixa em negrito por padrão.
  - `<td>`: São as células de dados comuns.

``` html

...

   <!-- o <table> é o elemento mais complexo do exemplo. -->
    <table border="1">
        <!-- <tr> (Table Row) cria a linha. Para pensarmos em colunas, temos que 
         antes pensarmos em linha, então, sempre terá uma linha -->
        <tr>
            <!-- <th> (Table Header) cria as celulas como cabeçalho. Dessa forma, os conteúdos dessas
             células serão renderizados em negrito e centralizados. -->
            <th>Dia</th>
            <th>Curso</th>
            <th>Nível</th>
            <th>Vagas</th>
        </tr>
        <tr>
            <!-- <td> (Table Data) preenche as células com o conteúdo informado. -->
            <td>15/05</td>
            <td>Saladas Tropicais</td>
            <td>Iniciante</td>
            <td>10</td>
        </tr>
        <tr>
            <td>22/05</td>
            <td>Molhos Naturais</td>
            <td>Intermediário</td>
            <td>08</td>
        </tr>
        <tr>
            <td>07/06</td>
            <td>Trabalhando com Ervas Finas</td>
            <td>Avançado</td>
            <td>05</td>
        </tr>
    </table>

...

```

## 5. Hiperlinks : O Elemento `<a>`
A tag `<a>` (âncora) transforma um texto em um portal para outro lugar:

- Atributo `href`: Indica a URL de destino.  
- Atributo `target="_blank"`: Este atributo instrui o navegador a abrir o link em uma nova aba. Isso é uma boa prática de UX (Experiência do Usuário), pois mantém o visitante no site da FPenha Corporation enquanto ele consulta referências externas.

``` html

...

    <p>
        Quer saber mais sobre nutrição? Visite o 
        <!-- <a> (Âncora): É o link que conecta nossa página ao mundo exterior.
         - O atributo href é o destino (onde o usuário será conduzido ao clicar).
         - O atributo target="_blank" informa ao navegador que ele deverá abrir o 
         link em uma nova aba, garantindo que o usuário não abandone seu site ao 
         clicar para pesquisar em outro lugar. -->
        <a href="https://www.who.int" target="_blank">Portal da Organização Mundial da Saúde</a>.
    </p>
    
...

```
